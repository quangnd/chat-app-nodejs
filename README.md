#  Chat application using Socket.IO ⛺️
See demo at [Socket.IO chat app](https://pure-woodland-28742.herokuapp.com/)

## Features

- [X] Separate room with username.
- [X] User online list.
- [X] Share your location.
- [X] Core chat.
- [ ] Save to DB.
- [ ] Testing.

## Up & Running

- Run `npm install`
- Run `npm run dev`
